School Management API

This is a simple backend project built using Node.js, Express, and MySQL.
It allows users to add schools and fetch a list of schools sorted based on distance from a given location.

Features
Add new school
Get schools sorted by distance
Basic validation
Tech Stack
Node.js
Express.js
MySQL
